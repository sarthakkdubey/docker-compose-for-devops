# Docker Compose for DevOps

This repository contains my Docker Compose practice projects created while learning DevOps and container orchestration concepts.

Each folder inside this repository represents a separate project or setup built using Docker Compose. The goal of this repository is to practice how multiple containers communicate with each other, how networking works, how volumes are managed, and how real-world multi-container applications are deployed.

The repository will continue to grow as I explore more Docker Compose concepts and build different multi-service applications.
